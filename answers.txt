Q1: Why might we consider using Git?

Answer: One reason is that Git allows you to work on a local project so that it doesn't ruin the stable version.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to healthcare. An example is having a large dataset of patient records and using the data to predict conditions the patient could develop.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging applies the changes made on another branch to the current branch, whereas rebasing "replays" the changes we made on another branch onto the current branch.
